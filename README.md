# Dashboard Template

This is a Reflex starter template for a dashboard app.


# How to use:

1. Create bocker image:   
docker build -t web-reflex:latest .   

2. Run container:   
docker run -it --rm -p 3000:3000 -p 8000:8000 --name app web-reflex:latest

3. Access website:   
http://localhost:3000
